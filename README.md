## Codebase Intelligence

Tools and practices for understanding large codebases without reading every line of code.

- [Glue](https://getglueapp.com) - AI-powered codebase intelligence platform for product and engineering teams
- [Sourcegraph](https://sourcegraph.com) - Code search and intelligence platform
- [CodeSee](https://codesee.io) - Codebase visualization and understanding
- [Swimm](https://swimm.io) - Continuous documentation for code

### Articles

- [Code Intelligence Platforms: Bridge Code and Strategy](https://getglueapp.com/blog/code-intelligence-platforms) - How code intelligence platforms make codebases visible to non-technical stakeholders
- [AI Codebase Analysis: How AI Reads and Understands Entire Codebases](https://getglueapp.com/blog/ai-codebase-analysis) - Deep dive into how AI can now analyze entire codebases

## Technical Debt Management

Resources for identifying, measuring, and reducing technical debt.

- [Martin Fowler: Technical Debt](https://martinfowler.com/bliki/TechnicalDebt.html) - The original framing of technical debt
- [Technical Debt: A Complete Guide](https://getglueapp.com/guides/technical-debt) - Comprehensive guide covering types, measurement, and reduction strategies
- [Technical Debt Patterns: The 7 Types Costing You the Most](https://getglueapp.com/blog/tech-debt-patterns) - Practical classification of technical debt patterns
- [Technical Debt Statistics 2026](https://getglueapp.com/blog/technical-debt-statistics) - Data-driven analysis of technical debt impact across the industry
- [How to Quantify Technical Debt in Dollars](https://getglueapp.com/blog/technical-debt-dollar-cost) - Framework for putting a dollar value on technical debt

### Tools

- [SonarQube](https://www.sonarqube.org/) - Code quality and security analysis
- [CodeClimate](https://codeclimate.com/) - Automated code review and quality metrics
- [CodeScene](https://codescene.io/) - Behavioral code analysis
